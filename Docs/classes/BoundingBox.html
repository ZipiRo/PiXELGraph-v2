BoundingBox class

Header: #include "Graphics/BoundingBox.h"

Axis-aligned rectangle defined by its edges, used for bounds checks and basic collision-style logic.

Example

// Creating and using a bounding box
BoundingBox box(10, 5, 30, 15);
Vector2 p(20.0f, 10.0f);

bool insideX = p.x >= box.left && p.x <= box.right;
bool insideY = p.y >= box.top  && p.y <= box.bottom;

if (insideX && insideY)
    Debug::Log("Point is inside the box\n");
