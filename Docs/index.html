PiXELGraph-v2 API Reference

PiXELGraph-v2 is a retro-style Windows console engine for simple 2D games, tools, and visual experiments. This site documents the C++ API exposed in the Include/ folder.

How to Use These Docs

  • Use the sidebar to pick a class.
  • On each class page you get:
    • A short description of what the class is for
    • All methods, with signatures and descriptions
    • Member variables, enums and related free functions
    • Optional global variables from the same header
    • A small usage example at the bottom
  • The README tab shows the high-level usage guide you would put into README.md on GitHub.

Key Engine Pieces

  • Core Engine: PiXELGraph, Scene, SceneManager, Time, View
  • Graphics: Screen, Shape, Rectangle, Elipse, Text, Font, Color, Vector2, Transform, BoundingBox, Vertex
  • Input & Window: Input, Event, Window
  • Audio: AudioClip, AudioSource
  • Debugging & Errors: Debug, Error

Quick Start Checklist

  1. Copy Include/, Library/, Resources/font.cf and DebugWindow.cpp into your project.
  2. Compile and link against PiXELGraph-v2.a and DebugWindow.cpp.
  3. Derive your engine class from PiXELGraph and call Init(...) in the constructor.
  4. Either:
    • Use the scene system (Scene, SceneManager), or
    • Disable scenes in EngineSettings.h and implement Start/Event/Update/Draw/Quit in your engine.
  5. Use Screen + shape/text classes to draw, and Input/Event for keyboard and mouse.
